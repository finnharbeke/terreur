# terreur
dans la pièce Terreur, le public devient jury au procès de Laura Koch, major de l'armée de l'air accusée d'avoir provoqué la mort de 164 passagers. Elle a abbattu un avion détourné par un terroriste quie menaçait de s'écraser sur un stade rassemblant 70 000 spectateurs. À la fin, le public doit fair un choix: condamner ou acquitter Laura Koch.
